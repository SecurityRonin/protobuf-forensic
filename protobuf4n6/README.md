# protobuf4n6

A protoscope-style schemaless protobuf CLI: dump a protobuf blob (from a file,
standard input, or a hex argument) as an indented field tree, JSONL, or a
protoscope-like format, with forensic timestamp flagging.

Part of [protobuf-forensic](https://github.com/SecurityRonin/protobuf-forensic).
