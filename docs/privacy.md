# Privacy Policy

*Last updated: 2026-07-11*

## Summary

protobuf-forensic is a local Rust library and CLI. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

protobuf-forensic decodes only the protobuf bytes you point it at (a file, standard input, or a hex argument). All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

protobuf-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

protobuf-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/protobuf-forensic](https://github.com/SecurityRonin/protobuf-forensic).
